# 虛擬擲筊模擬器 - 神明顯靈版 ⛩️

這是一款基於 Python 與 Pygame 開發的「虛擬擲筊模擬器」。專案中結合了簡單的物理碰撞、動態數學音效合成（免外部音效檔）以及神光與土地公插畫淡入特效，讓你隨時隨地都能誠心向神明祈求指點迷津！

---

## ✨ 遊戲特色

* **寫實動態效果：** 模擬雙手捧筊、拋出、空中旋轉與地面邊界反彈的物理動態。
* **神明視覺顯靈：** 擲筊結果揭曉時，會伴隨旋轉的金色神光背景與土地公插畫淡入，並具微弱的上下浮動呼吸感。
* **動態音效合成：** 內建「木頭敲擊聲」與「咻咻破風聲」的數學訊號合成器。即便資料夾內沒有音效檔，程式也能動態生成流暢的二進位音效。
* **防呆視覺排版：** 最佳化的文字排版與右上角輕量化重來提示，完全不遮擋神明慈祥的插畫。

---

## 🎮 操作說明

1. 開啟程式後，雙手會捧著筊杯在畫面下方。
2. 內心誠心祈求想問的事項。
3. 按下 **[ 空白鍵 (Space) ]** 擲筊。
4. 觀看神明給予的指示：
   * 🌟 **【 聖 筊 】**（一平一凸）：神明聽見了，同意你的請求！
   * 😂 **【 笑 筊 】**（兩片皆平）：神明笑而不答，換個方式問問看吧。
   * 🛑 **【 陰 筊 】**（兩片皆凸）：神明不認同，建議重新思考。
5. 在結果畫面按下 **[ 空白鍵 (Space) ]**，即可將筊杯重新收回手中，再次祈求。
